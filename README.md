# Android-Pile
##简要:
  集合网易头条新闻,开眼视频,知乎日报...etc的App;
###Use
<ul>
<li>ExoPlayer播放视频</li>
<li>OkHttp3网络请求</li>
<li>ActiveAndroid ORM</li>
<li>Glide图片加载</li>
<li>Gson Json数据解析</li>
</ul>
